# Suspend-Mumu

## 介绍

公主连结 pcr 一键挂起星云引擎黄MuMu进程，防止滑刀

附带黄MuMu 999次极速连点配置文件

## 使用方法

开启程序后使用F2暂停/恢复黄MuMu进程

## 注意事项

只有星云引擎的黄MuMu才可以使用该程序，标准引擎或者其他模拟器不适用，可以自行修改 `SuspendMuMu/KeyboardHook` 150行进行适配
